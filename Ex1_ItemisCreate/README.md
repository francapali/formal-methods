# Importing the Project into _itemis CREATE_

This guide provides instructions for importing the project into the itemis CREATE environment.

---

## Requirements
* **Java JDK**: A compatible version is required (e.g., JDK 11).
* **itemis CREATE**: Ensure the software is installed and updated.

---

## Import the project
1.  **Open** itemis CREATE.
2.  Go to **File > Import...**
3.  Choose **General > Existing Projects into Workspace** and click **Next**.
4.  Select **Select root directory** and browse to the project folder.
5.  Select the discovered projects and click **Finish**.
